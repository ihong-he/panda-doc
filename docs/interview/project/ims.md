# 博欣信息管理系统（ERP系统）

项目概述
博欣信息管理系统（ERP系统）基于vue3+vite构建，包含PC端和移动端，集成制造研发类企业日常工作管理，包含进销存、产品管理、物料管理、生产制造等模块，快速提供决策信息，提升企业的营运绩效与快速反应能力。
:::


## 一、项目技术

vue3、vite、pina、vue-router、element-plus、axios、uniapp、uview、echarts、signalr消息推送及自定义组件等

## 二、我的工作

- 用户管理：用户管理、角色管理、权限管理
- 订单管理：采购订单和销售订单
- 产品管理：物料管理、产品管理
- BOM管理：包含BOM树的增删改查、导入、导出、批量操作以及版本控制等。
- 生产管理：生产计划、生产任务、生产进度、生产报工
- 仓库管理：仓库库存、仓库出库、仓库入库
- 客户及供应商管理：客户管理、供应商管理

## 三、难点


### （一）大数据量表格渲染卡顿

> BOM树中的数据量可能非常大，因此需要考虑如何处理超大数据量。

使用虚拟列表组件el-table-v2，可以轻松实现超大数据量的渲染。

### （二）图纸大文件上传与预览

> 图纸上传和预览，需要考虑如何处理大文件上传和图纸预览。

使用切片上传和断点续传，可以轻松实现大文件上传。
图纸预览使用kkFileView


### （三）工序及工艺流程

> 一个工艺流程由不同的工序组成，需要考虑如何实现可视化编辑。

将流程图结构序列化为 JSON 数据，通过接口提交至后端；前端使用 logicflow 扩展插件实现图形化编辑与事件绑定。

### （四）用户权限

> 菜单权限和按钮权限管理，需要考虑如何实现。


## 四、项目亮点

高可用性、高性能、高安全性、强扩展性
